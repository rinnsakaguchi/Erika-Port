Panduan Instalasi

  1. Backup semua data penting Anda
  2. Unlock bootloader perangkat Anda
  3. Install custom recovery (TWRP recommended)
  4. Download file ROM yang sesuai
  5. Flash melalui custom recovery
  6. Clear cache/dalvik
  7. Reboot sistem
